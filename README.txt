===================================================
       GROUPLANCING BROWSER - INSTALLATION
===================================================

QUICK START:
1. Extract this folder to: C:\Program Files\GroupLancing\
2. Double-click: scripts\launch.bat
3. Enter your license key on first run
4. Browser opens with secure proxy!

SYSTEM REQUIREMENTS:
- Windows 10 or later (64-bit)
- 700 MB free disk space
- Internet connection

FEATURES:
✓ VLESS/VMESS proxy support
✓ License key protection
✓ Locked proxy (cannot be changed)
✓ Firefox with restricted settings
✓ Zero configuration needed

FIRST TIME:
- launch.bat will ask for your license key
- Key is saved automatically
- Browser opens with proxy enabled

NEXT TIME:
- Just run launch.bat
- No prompts, everything automatic

TROUBLESHOOTING:
If browser doesn't open:
1. Check internet connection
2. Make sure you entered license key correctly
3. Make sure port 9050 is not blocked by firewall
